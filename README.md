 Project Title

Laravel API CALL

 Project Description

This project is a Laravel-based API application designed for testing different HTTP methods through Postman. It supports CRUD functionalities including GET, POST, PUT, PATCH, and DELETE requests connected to a database system.

 Setup Instructions

1. So, the first step is to Launch XAMPP and start both Apache and MySQL services (using port 3306).

2. Open the project folder using Visual Studio Code.

3. Open the terminal inside VS Code.

4. Execute the following commands:

php artisan migrate
php artisan serve

5. Open Postman to access and test the API routes.

6. Perform API requests such as GET, POST, PUT, PATCH, and DELETE.
